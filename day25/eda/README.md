# Day 25 - AM - Exploratory Data Analysis

## Part A - Concept Application

This task deals with analysis of the Titanic Dataset which focuses on understanding the dataset through exploration and statistical summary. Descriptive statistics provide insight into numerical features, while categorical analysis helps understand distributions in categorical features.

## Key Learnings

- Missing values can significantly affect results and must be treated carefully.
- Numerical statistics help understand central tendency, spread, and possible outliers.
- Categorical frequency analysis reveals dominant groups and class imbalance.
- Visualizations make patterns easier to interpret.
- Proper preprocessing improves reliability of later machine learning steps.
